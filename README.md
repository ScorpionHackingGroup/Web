# ScorpionYug — Personal Blog & Portfolio

A modern, responsive Jekyll blog/portfolio site for **scorpionyug.my.id**, hosted free on GitHub Pages.

## Features

✅ **Built with Jekyll** — official GitHub Pages framework
✅ **Auto-generated sitemap** via jekyll-sitemap plugin
✅ **SEO optimized** — jekyll-seo-tag, Open Graph, Twitter cards
✅ **Responsive design** — mobile, tablet, desktop
✅ **Dark/Light theme** — auto-detects + manual toggle
✅ **RSS feed** — jekyll-feed plugin
✅ **Pagination** — jekyll-paginate
✅ **Search-friendly** — proper meta tags, structured data
✅ **Fast** — static site, no JS frameworks, optimized CSS
✅ **Easy to customize** — change colors, fonts, layout

## Pages

- 🏠 `/` — Homepage with hero + latest posts
- 📝 `/blog/` — All blog posts
- 🐛 `/writeups/` — Bug bounty writeups (filter by category)
- 👤 `/about/` — About me
- 📧 `/contact/` — Contact info
- 🔒 `/404.html` — Custom 404 page
- 🤖 `/sitemap.xml` — Auto-generated by plugin
- 📡 `/feed.xml` — RSS feed

## Quick Start

### Local development

```bash
# 1. Install Jekyll
gem install bundler jekyll

# 2. Install dependencies
cd scorpionyug.github.io
bundle install

# 3. Run dev server
bundle exec jekyll serve
# → http://localhost:4000
```

### Add a new post

Create a file in `_posts/` with this format:

```markdown
---
layout: post
title: "Your Post Title"
date: 2026-06-15 10:00:00 +0530
categories: [bug-bounty, recon]
description: "Brief description for SEO"
author: "ScorpionYug"
image: /assets/images/posts/your-image.jpg
---

Your content here in Markdown.
```

### File naming convention

```
_posts/YYYY-MM-DD-your-post-title.md
```

Example:
```
_posts/2026-06-15-my-new-writeup.md
```

## Customization

### Change colors

Edit CSS variables in `assets/css/main.css`:

```css
:root {
  --color-accent: #b8472a;      /* Change brand color */
  --color-accent-hover: #9a3a22;
  --font-display: 'Sora', sans-serif;
}
```

### Change site info

Edit `_config.yml`:

```yaml
title: "YourName"
tagline: "Your tagline"
email: "your@email.com"
url: "https://yourdomain.com"
```

### Add your avatar

Place image at `assets/images/avatar.jpg` and update `_includes/header.html` to use it.

## Deployment to GitHub Pages

### Initial setup

```bash
# 1. Create repo on GitHub
# → scorpionyug/scorpionyug.github.io (public)

# 2. Initialize git
git init
git add .
git commit -m "Initial commit"

# 3. Add remote
git remote add origin https://github.com/scorpionyug/scorpionyug.github.io.git
git branch -M main
git push -u origin main
```

### Auto-deploy

GitHub Pages automatically builds your site on every push to `main` branch.

**URL:** https://scorpionyug.github.io (or custom domain if configured)

### Custom domain (scorpionyug.my.id)

1. **DNS** — Add 4 A records to GitHub IPs:
   - 185.199.108.153
   - 185.199.109.153
   - 185.199.110.153
   - 185.199.111.153

2. **GitHub** — Settings → Pages → Custom domain: `scorpionyug.my.id`
3. **Enforce HTTPS** ✅
